# Getting Started

1. Configure environment variables

```bash
SPOTIFY_CLIENT_ID="SPOTIFY_CLIENT_ID"
SPOTIFY_CLIENT_SECRET="SPOTIFY_CLIENT_SECRET"
```

2. Package installation

```bash
npm i
```

3. Start server and run app

```bash
npm run dev
```
